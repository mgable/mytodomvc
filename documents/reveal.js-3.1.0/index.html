What is End to End (E2E) testing

  • is not unit testing
  • tests a complete workflow
  • based on performing an action and testing the results
  • user centric
  • primary way to regression test

What is Protractor.js

  • Protractor is an end-to-end test framework for AngularJS applications.
  • built on Selenium / Webdriver
  • uses Jasmine syntax
  • based on asynchronous programming model

Developing an E2E testing strategy

  • determine what to test
  • determine when to write the test
  • determine aspects of behavior to test

Types of E2E testing

  • Rendering
  • Behavioral
  • Data
  • Performance

Render Tests

if a element rendered on the page

Behavior Tests

if the element behaves in the intended way

Data Tests

if content matches the data

Performance Tests

if the response times are within acceptable limits

The Tests

  • written using the Jasmine syntax
  • asynchronous programming using promises

Jasmine Syntax

  • describe the test
  • it should have specifications
  • expect the specifications to have some result

Jasmine for Test 1.0

  • describe enter a todo
  • it should have an input field
  • expect the input field to be present
  • it should have an input field that accepts user data
  • expect data to be entered into the input field
